# Instalación

Clonar el repositorio:

```bash
git clone https://github.com/Mgroyes/Fractal-estudio-digital.git
```

Entrar al proyecto:

```bash
cd Fractal-estudio-digital
```

Instalar dependencias:

```bash
npm install
```

Iniciar servidor de desarrollo:

```bash
ng serve
```

Abrir en:

```bash
http://localhost:4200/
```

---

# Scripts disponibles
